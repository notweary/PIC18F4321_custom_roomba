#Description
very simple roomba; it detects objects 12 cm infront, if so, changes direction.
Attached dustpan below with rotating brushes at its side to sweep trash into pan
#equipment
HC-SR04 ultrasonic sensor
PIC18F4321 mcu
DC electric motors
L298N motor drive controller
HD44780 1602 LCD Display
TIP122 NPN Transistor
12V 2400 mAh rechargeable Li-ion battery

Programed with MPLAB IDE, xc8 compiler, and Pickit3
